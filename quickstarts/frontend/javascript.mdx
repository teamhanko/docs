## Setup your Hanko project
Go to the [Hanko console](https://cloud.hanko.io/) and [create a project for this application.](/setup-hanko-cloud)

<Note>
  During creation make sure to input the URL you will be developing on as the `APP URL`.

  Keep in mind that during development the url you are working on cant include an ip.
  If you use the Live Server extention make sure you are testing at `localhost:PORT` instead of `ip:PORT`.

  (Most likely your URL will be http://localhost:5000/) \
\
  If you already have a project you can change the URL at:\
  `'your project' > Settings > URLs > App URL`.
</Note>


## Add the Hanko API URL

Retrieve the API URL from the [Hanko console](https://cloud.hanko.io/) and place it in a config file..

```js config.js
  export const config = {
      HANKO_API_URL: 'https://45449ca9-9bc5-4304-81dd-fa31e06b4190.hanko.io'
  }
```

<Note>
  If you are self-hosting you need to provide the URL of your running Hanko
  backend.
</Note>


## Add `<hanko-auth>` component

The `<hanko-auth>` web component adds a login interface to your app. Begin by importing the `register` function from `@teamhanko/hanko-elements`. Call it with the Hanko API URL as an argument to register `<hanko-auth>` with the browser's [CustomElementRegistry](https://developer.mozilla.org/de/docs/Web/API/CustomElementRegistry). Once done, use it in your markup.

```html index.html
<body>
  <hanko-auth> </hanko-auth>

  <script type="module">
      import { config } from './config.js';
      import { register } from "https://esm.run/@teamhanko/hanko-elements";
  
      const { hanko } = await register(config.HANKO_API_URL);
  
      hanko.onSessionCreated(() => {
        // successfully logged in, redirect to a page in your application
        document.location.href = "/dashboard.html";
      });
    </script>
    
</body>
```

## Add `<hanko-profile>`

The `<hanko-profile>` component provides an interface, where users can manage their email addresses and passkeys.

```html dashboard.html
<body>

    <hanko-profile> </hanko-profile> 

    <script type="module">
        import { config } from './config.js';
        import { register } from "https://esm.run/@teamhanko/hanko-elements";
```

## Implement logout functionality

Use the Hanko client provided by `@teamhanko/hanko-elements` to log out users. On logout a custom event is
dispatched that you can subscribe to:

```html dashboard.html
<body>

    <hanko-profile> </hanko-profile> 
    <button id="logout-button">Log Out</button>

    <script type="module">
        import { config } from './config.js';
        import { register } from "https://esm.run/@teamhanko/hanko-elements";
    
        const { hanko } = await register(config.HANKO_API_URL);

        document.getElementById("logout-button")
        .addEventListener("click", () => {
            console.log("test");
            hanko.user.logout();
        });

        hanko.onUserLoggedOut(() => {
            // successfully logged out, redirect to a page in your application
            document.location.href = "/"
            console.log("test");
        })
    </script>
</body>
```
