title: FIDO UAF Client for iOS
description: Add on-device FIDO UAF authentication to your iOS app.

You can integrate FIDO-based strong multi-factor authentication into your existing iOS app using the open source Hanko FIDO UAF Client.

To learn more about integrating the client, view the documentation on [GitHub](https://github.com/teamhanko/fidouafclientios) or get started with the [quickstart guide](/hanko-authenticator/quickstart/uaf-client).
